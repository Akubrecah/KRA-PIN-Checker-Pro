Introduction

The AkubrecaH KRA API allows you to programmatically verify reliable taxpayer information. We currently support checks via National ID and KRA PIN number.

Authentication

All API requests must include your API Key in the `Authorization` header. 

Authorization: Bearer YOUR_API_KEY

Verify by PIN

Retrieve taxpayer details using their KRA PIN.

GET /api/v1/verify/pin/{pin}

Request Example


const response = await fetch('https://api.akubrecah.com/v1/verify/pin/P051234567Z', {
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
    }
});
const data = await response.json();

Response Example


{
    "status": "success",
    "data": {
        "taxpayer_name": "JANE DOE",
        "pin": "P051234567Z",
        "tax_station": "West of Nairobi",
        "status": "Active"
    }
}
