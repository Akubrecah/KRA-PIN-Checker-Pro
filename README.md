# KRA PIN Checker Pro - PDF Generator

A verification suite and certificate generator for KRA PINs.

## Features
- **PIN Verification**: Validate KRA PINs and retrieve taxpayer details.
- **Certificate Generation**: Generate official-looking A4 PIN certificates.
- **Modern UI**: Clean, responsive interface with easy navigation.
- **Standalone PDF Engine**: Robust, client-side PDF generation using `html2pdf.js`.

## Setup
1. Open `index.html` in any modern web browser.
2. No server installation required for the frontend.

## Usage
1. Select "Verify by PIN" or "Verify by ID".
2. Enter the taxpayer details.
3. Click "Generate PIN Certificate" to preview and download the PDF.
